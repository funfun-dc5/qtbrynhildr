クライアントのインストールと設定 (Mac版)
============================================

1. ダウンロード
----------------

.dmgファイルをマウントし、QtBrynhildr.app を /Applications にコピーしてください。


2. アンインストール
----------------------------------

/Applications/QtBrynhildr.app を削除してください。

3. 起動/終了と設定ファイル
--------------------------

* 設定ファイル

  標準では以下のファイルに保存されます。 ::

   ~/.config/mcz-xoxo/Qt Brynhildr.ini

  オプションにファイル名を与えることで変更できます。
