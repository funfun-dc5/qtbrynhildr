ソースコードの公開
===================

 C++言語素人, Windows/Linuxソフトウェア開発素人, Qt素人の拙いソースコードではありますが、
 恥ずかしながらGitHubで公開してしまおうと思っています。 [#]_ 

 綴りが間違ってる、インデントが揃ってない、など細かい所でもフィードバック頂けたら幸いです。

 Qt Brynhihldrのソースコードは基本的にQt5にほぼ依存しています。Qt5のサポートする環境であれば
 移植可能"かも"しれません。Qt5のサポートする環境については以下のページを御覧ください。

 <http://doc.qt.io/QtSupportedPlatforms/index.html>

 詳細は別途ブログに投稿予定です。

 ソースコードは以下で公開しています。

 <https://github.com/funfun-dc5/qtbrynhildr.git>
