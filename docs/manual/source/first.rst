最初に
=======================

 離れた場所や別の部屋にあるコンピュータを操作したいと思うことが時々あります。
 昔は、目の前にあるパソコンだけがすべてでしたが今は違いますね。

 かなり昔から、離れた場所にあるコンピュータを操作するという技術は存在しました。
 専用線をわざわざ引いたり、電話回線を利用したりして。

 今はほとんどすべてのコンピュータが高速なネットワークに繋がっていて、相互にやりとりしています。
 あるキーを打ったという単純な情報を送ることさえ大変だった昔に比べると雲泥の差に驚くばかりです。

 この高速なネットワークを利用して、さも目の前に使いたいコンピュータが存在するかのように画面を表示し、
 操作できる技術があります。その技術を総称して"リモートデスクトップ"と呼ぶようです。
 個人的には「Windows的」な用語かなという印象はありますが…

 Windows PCを操作することに特化したリモートデスクトップソフトウェアの1つとして **Brynhildr**
 というソフトウェアがフリーで公開されています。

 別のWindows PCから操作することはもちろん、iPhoneといったiOS機器からも
 **"遠隔で"** 操作できるそうです。

 **Brynhildr** を利用して、Windows PCやiPhone以外のいろいろな機器からも操作できたら **"面白いかも"**
 というのが最初の単純な発想でした。

 2024.7.22より **Brynhildr** は **シェアウェア** となりました。
