動作環境

サーバ
------

 Brynhildrのページの **SystemRequirements** を参照してください。

クライアント
------------

(1) Windows版  [#]_ 
^^^^^^^^^^^^^^^^^^^^
  * Windows XP      (64bit/32bit) (Qt5.6以下でビルド)
  * Windows Vista   (64bit/32bit) (Qt5.6以下でビルド)
  * Windows 7       (64bit/32bit)
  * Windows 8       (64bit/32bit)
  * Windows 8.1     (64bit/32bit)
  * Windows 10      (64bit/32bit)

 .. [#] すべてのエディションで確認したわけではありません。

(2) Linux版
^^^^^^^^^^^^^
  * Ubuntu 14.04 LTS (x64/x86)以降
  * CentOS 7 (x64)

(3) Mac版
^^^^^^^^^^
  * Mac OS X 10.10 (Yosemite)以降

(4) FreeBSD版
^^^^^^^^^^^^^
  * FreeBSD 10.2 (x64/x86)
