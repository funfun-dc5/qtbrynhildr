クライアントのインストールと設定 (Mac版)
============================================

1. ダウンロード
-----------------------------

 以下からダウンロードします。

 <https://github.com/funfun-dc5/qtbrynhildr/releases>

2. インストール
----------------
 Finderで.dmgファイルをダブルクリックしてマウントし、Qt Brynhildr.app を /Applications にコピーしてください。

3. アンインストール
-------------------

/Applications/Qt Brynhildr.app と設定ファイルを含むディレクトリ(~/.config/mcz-xoxo)を削除してください。

4. 起動/終了と設定ファイル
--------------------------

 Launchpad から Qt Brynhildr.app をクリックすることで起動します。

* 設定ファイル

  標準では以下のファイルに保存されます。 ::

   ~/.config/mcz-xoxo/Qt Brynhildr.ini

  オプションにファイル名を与えることで変更できます。
