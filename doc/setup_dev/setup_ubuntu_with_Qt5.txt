---------------------------------------------------------------------------
 Setup Development Environment for Qt Brynhildr on Ubuntu
---------------------------------------------------------------------------

1) install Qt5 packages

 $ sudo apt-get install qt5-default qtmultimedia5-dev

 (*) You may have to do "sudo apt-get install libpulse-dev" for Qt5.9.0.

2) run qmake

 $ cd qtbrynhildr/src
 $ qmake qtbrynhildr.pro

3) build

 $ make
