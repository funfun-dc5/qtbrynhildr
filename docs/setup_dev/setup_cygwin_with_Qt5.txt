---------------------------------------------------------------------------
 Setup Development Environment for Qt Brynhildr on Cygwin
---------------------------------------------------------------------------

1) install Qt5 packages

 ex.) run setup-x86_x64.exe

 libQt5Core-devel
 libQt5Multimedia-devel
 libQt5X11Extras-devel
 libGL-devel

2) run qmake in Cygwin Terminal

 $ cd qtbrynhildr/src
 $ ./setup cygwin

3) build

 $ make
