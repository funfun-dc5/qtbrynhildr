---------------------------------------------------------------------------
 Setup Development Environment for Qt Brynhildr on MacOSX (10.10 or later)
---------------------------------------------------------------------------

1) install Xcode

Install Xcode-7.2.1 at App Store.

2) install Qt5

for example)

 http://download.qt.io/archive/qt/5.9/5.9.0/qt-opensource-mac-x64-5.9.0.dmg

3) check setup script for Qt5 (src/setup_qt5)

4) run src/setup

 cd qtbrynhildr/src
 ./setup

5) make
