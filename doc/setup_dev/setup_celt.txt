---------------------------------------------------------------------------
 Setup CELT
---------------------------------------------------------------------------

1) download CELT library v0.11.1 source code

 http://downloads.xiph.org/releases/celt/celt-0.11.1.tar.gz

 ( http://www.celt-codec.org/ )

2) build static library (libcelt0.a)

 configure script option : --enable-custom-modes --disable-oggtest

3) copy header files and static library

 copy celt.h, celt_types.h and libcelt0.a(named "libcelt.a") to qtbrynhilr/libs/celt/

4) build Qt Brynhildr
