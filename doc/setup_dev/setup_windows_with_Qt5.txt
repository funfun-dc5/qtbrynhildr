---------------------------------------------------------------------------
 Setup Development Environment for Qt Brynhildr on Windows.
---------------------------------------------------------------------------

0) install Qt5.4.2 (for MinGW-4.9.1) and MinGW-W64 and Cygwin

 http://download.qt.io/archive/qt/5.4/5.4.2/qt-opensource-windows-x86-mingw491_opengl-5.4.2.exe

1) check setup script for Qt5 (src/setup_qt5)

2) run src/setup

 cd qtbrynhildr/src
 ./setup

3) make

 (*) If you use both of cygwin and MinGW and Qt, check PATH environment variable.
    PATH order : Qt -> MinGW -> cygwin

for example)

/c/Tools/develop/Qt/Qt5.4.2/5.4/mingw491_32/bin:/c/Tools/develop/Qt/Qt5.4.2/Tools/mingw491_32/bin:/c/Tools/develop/cygwin/bin

NOTE)

(1)
If you set wrong PATH order, you will see no message. 
The program might stop in startup before reached main function.

If you see a message, check PATH order.(^_^;

(2)
qtbrynhildr.pro and Makefile* has CR-LF code (NOT LF).
