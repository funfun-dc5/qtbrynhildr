---------------------------------------------------------------------------
 Setup Development Environment for Qt Brynhildr on Linux (Ubuntu-14.04 LTS)
---------------------------------------------------------------------------

1) install Qt5.4.2 and g++

 [64bit]
 http://download.qt.io/archive/qt/5.4/5.4.2/qt-opensource-linux-x64-5.4.2.run

 [32bit]
 http://download.qt.io/archive/qt/5.4/5.4.2/qt-opensource-linux-x86-5.4.2.run

 chmod 755 qt-opensource-linux-x64-5.4.2.run
 sudo ./qt-opensource-linux-x64-5.4.2.run

 (*) If you don't have g++ command, do "sudo apt-get install g++"
     and you may have to do "sudo apt-get install libglu1-mesa-dev" for openGL.

2) check setup script for Qt5 (src/setup_qt5)

3) run src/setup

 cd qtbrynhildr/src
 ./setup

4) make
