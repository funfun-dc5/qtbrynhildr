---------------------------------------------------------------------------
 Setup Development Environment for Qt Brynhildr on Linux (Ubuntu-14.04 LTS)
---------------------------------------------------------------------------

0) install Qt5.5.0 and g++

 http://download.qt-project.org/official_releases/qt/5.5/5.5.0/qt-opensource-linux-x64-5.5.0.run

 chmod 755 qt-opensource-linux-x64-5.5.0.run
 sudo ./qt-opensource-linux-x64-5.5.0.run

 (*) If you don't have g++ command, do "sudo apt-get install g++"
     and you may have to do "sudo apt-get install libglu1-mesa-dev" for openGL.

1) check setup script for Qt5 (src/setup_qt5)

2) run src/setup

 cd qtbrynhildr/src
 ./setup

3) make
